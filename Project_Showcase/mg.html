Details

This is a Python-based memory card game implemented using the graphics library. The game requires the player to match pairs of cards with identical symbols. It features different game modes (Images Mode or Numbers Mode) and difficulty levels (Easy, Normal, and Hard). The player earns keys by successfully matching pairs, aiming to collect all keys within the least moves and time possible.  To learn more! Visit Project's  Github 

Also you can visit my personal Portfolio 

Memory Game:

- Game Modes: Players can choose between Images Mode or Numbers Mode.
- Difficulty Levels: Three difficulty levels are available: Easy, Normal, and Hard.
- Interactive Interface: The game provides a user-friendly graphical interface for seamless gameplay.
- Sound Effects: Interactive sound effects enhance the gaming experience, including background music and click sounds.
- Pause Functionality: Players can pause the game at any time and resume later.
- Scoreboard: After completing the game, players can view their performance and rankings.
- Exit Button: Allows players to exit the game at any time.

Requirements:

- Python (for running the script)
- `graphics`, `random`, `time`, `platform`, `pygame`, `tkinter` libraries
- Sound files in `mp3` format for background music and sound effects
- Image files in `png` format for graphics

File Structure

  • main.py: Python script containing the game implementation.
  • bg.png: Background image file.

  • cardback.png: Image file for the back of the card.

  • card_close.mp3: Sound file for card closing.
  • click_sound.mp3: Sound file for click interactions.

Technical Details

Function Definitions

  • Pause_function(win): Pauses the game and displays a message. Resumes the game upon user interaction.
  • quit_game(win): Closes the game window and exits the program.
  • get_user_response(response_var, user_response): Sets the user's response and closes the dialogue box.
  • ask_yn(ynMessage): Displays a Yes/No dialogue box and returns the user's response.
  • ExitButton(win): Displays an exit button on the game window. Closes the game upon clicking the button.
  • sound_button_function(): Toggles sound effects on/off and displays corresponding messages.
  • welcome_music(): Plays background music upon starting the game.
  • click_music(): Plays a click sound effect upon user interaction.
  • card_music(): Plays a sound effect when a card is opened.
  • card_close(): Plays a sound effect when a card is closed.
  • OsCheck(): Checks if the operating system is Windows.
  • Welcome(win): Displays the welcome screen and prompts the user to start the game.
  • GameModePick(win): Allows the user to select the game mode (Images Mode or Numbers Mode).
  • Difficult(win): Allows the user to select the difficulty level (Easy, Normal, or Hard).
  • Init(win): Initializes the game by prompting the user to enter their name and starting the timer.
  • ReadPicturesNames(t, gamemode): Returns the file name of the card image based on the game mode and card type.
  • DrawPictures(path, p, win, gamemode): Draws card images on the game window.
  • DrawText(p, win, num_list, clicked_card): Draws text on the card image.
  • ScoreBoard(win, move, key, mouse): Displays the scoreboard showing the number of moves, keys collected, and mouse clicks.
  • GameBoard(win, username, gamemode, difficult): Draws the game board with card images based on the selected game mode and difficulty level.
  • DecideWhichCard(x, y, card_point1, card_point2 , difficult): Determines which card is clicked based on mouse coordinates.
  • HasItBeenClickedBefore(clicked_card, exposed): Checks if a card has been clicked before.
  • CheckMacthes(c1, c2, num_list): Checks if the symbols on two cards match.
  • Reuslt(win, gametime, username, difficult): Displays the game result including player time and rankings.
  • InsertInsideTheTopPlayer(username, gametime, move, difficult): Inserts the player's performance into the top player list.
  • Main(): Main function that orchestrates the entire gameplay loop.
